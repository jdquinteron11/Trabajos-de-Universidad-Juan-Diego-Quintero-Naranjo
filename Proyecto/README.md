# Proyecto Final - Programación Orientada a Objetos

## Descripción

Proyecto final que integra todos los conceptos aprendidos durante el curso de POO.

## Objetivos

- Aplicar los principios de POO en un proyecto real
- Demostrar comprensión de encapsulamiento, herencia, polimorfismo y abstracción
- Implementar un diseño de software robusto y mantenible

## Estructura del Proyecto

```
Proyecto/
├── README.md           # Este archivo
├── docs/               # Documentación del proyecto
│   ├── diagramas/      # Diagramas UML
│   └── manual.md       # Manual de usuario
├── src/                # Código fuente
│   ├── models/         # Clases del modelo
│   ├── controllers/    # Controladores
│   └── views/          # Interfaces de usuario
└── tests/              # Pruebas unitarias
```

## Requisitos

1. **Diseño:** Debe incluir diagramas UML (clases, secuencia, etc.)
2. **Implementación:** Código limpio y bien documentado
3. **Pruebas:** Incluir pruebas unitarias cuando sea posible
4. **Documentación:** Manual de usuario y documentación técnica
